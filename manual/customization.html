Customization

TeXnicCenter allows you, to customize nearly every aspect of the graphical user interface (GUI).

To customize the user interface of TeXnicCenter, select in the menu 'Tools' the menu item 'Customize...'. This will open the dialog 'Customization'.

  • Specify, which commands should be available via the Menus and Toolbars;
  • customize the Toolbars;
  • customize the menu Tools;
  • customize the Keyboard Sshortcuts used to access commands using the keyboard;
  • customize the Menus;
  • customize the Text Format of the editor, the navigator and the output bar;
  • customize the general Look & Feel.

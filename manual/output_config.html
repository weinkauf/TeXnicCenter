Configuration of Output Profiles

There are two ways to configure output profiles:

  • One way, and that ist the simple one, (especially if MikTeX is installed), is to use the Ouput Wizard. This wizard will start automatically, if TeXnicCenter is started for the very first time.
  • The other way is to manually configure output profiles. This requires more work, but on the other hand provides more flexibility.

All output profiles, generated by the output wizard, can later be edited manually, using the output configuration.

If MikTeX is installed, the Output Wizard should be prefered. He detects a installation of MikTeX on the system and generates output profiles for DVI, PDF and postscript accordingly.

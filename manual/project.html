Project Management

TeXnicCenter provides a project management tool that helps to keep the files of one document together and to generate output for a document.  It is especially useful when working on large documents.

A project file will be generated for each project containing the project's properties. The stored information is:

  • The path to the main file of the project, relative to the directory the project file is placed in.  (This file will be passed to the LaTeX compiler.)
  • A flag that specifies whether the current project uses BibTeX.
  • A flag that specifies whether the current project uses MakeIndex.

Some features are only available if a project is opened.  Examples of these are Structure View (the navigator bar on the left side of the main window) and the automatic invocation of BibTeX and MakeIndex

Using projects is very simple. Just create a new project before starting to work on a new document. If the requirements for the project have changed, it is easy to simply modify the project's properties.

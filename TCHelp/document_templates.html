Document Templates

TeXnicCenter provides a powerfull feature called document templates.

A document template is a simple file or a wizard, that generates a frame for a new document. If you are having document frames, you are using again and again (i.e. a letter), then the document templates will be a big help for you.

TeXnicCenter knows to types of document templates:

  • File based document templates and
  • Document wizards.

You can specify several directories, that are containing document templates in the path options. The directories specified there should contain sub directories, which should represent the categories of the templates, located in those sub folders.

When creating a new project, TeXnicCenter provides the found document templates on the left side of the 'New Project' dialog.

The tabs are representing the sub folders of the directories, specified in the path options. If one or more directories are containing the same sub folders, TeXnicCenter will displayed the files, found in those sub folders, together in one list.

An Example:

Lets assume, you have specified to template directories in the path options:

  1. A local directory that keeps your private templates and
  2. a directory on your companies network, that contains company specific templates.

Your company's directory contains the following subfolders:

  • General
  • Articles
  • Letters

Whereby the directory "Letters" contains the following templates:

  • Order.tex
  • Notice.tex

Your local directory contains the following subfolders:

  • Letters
  • University

Whereby the directory "Letters" contains the following templates:

  • Love Letter.tex

If you are creating a new project now, TeXnicCenter will analyse the directories specified in the path options. The sub folders of the template directories listed there will become categories, which are represented by the tabs on the left side of the 'Project New' dialog and the files contained in the sub folders will be listed, when choosing the belonging tab.

For our example, the available tabs would be

  • Articles
  • General
  • Letters
  • University

The categories "Articles" and "General" would contain the templates from the company's directory. The category "University" would contain the templates from your local drive. And the category "Letters" would contain the templates from the company's directory merged with the one's from your local drive, which are:

  • "Order" (company/Letters/Order.tex)
  • "Notice" (company/Letters/Order.tex)
  • "Love Letter" (local/Letters/Love Letter.tex)

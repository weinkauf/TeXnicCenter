Document Templates

TeXnicCenter provides a powerfull feature called document templates.

A document template is a simple file or a wizard, that generates a frame for a new document. When using document frames over and over again (i.e. a letter), then using document templates will be a big help.

TeXnicCenter knows those types of document templates:

  • File based document templates and
  • Document wizards.

You can specify several directories, that are containing document templates in the path options. The directories specified there should contain sub directories, which should represent the categories of the templates, located in those sub folders.

When a new project is created, TeXnicCenter provides the document templates found on the left side of the dialog 'New Project'.

The tabs are representing the sub folders of the directories, specified in the path options. If one or more directories contain the same sub folders, TeXnicCenter will displaye the files found in those sub folders together in one merged list.

Example:

Lets assume that in the path options two template directories were specified:

  1. A local directory that keeps private templates and
  2. a directory on the companies network, that contains company specific templates.

The local directory contains the following subfolders:

  • Letters
  • University

Whereby the directory "Letters" contains the following templates:

  • Love Letter.tex

The company's directory contains the following subfolders:

  • General
  • Articles
  • Letters

Whereby the directory "Letters" contains the following templates:

  • Order.tex
  • Notice.tex

When creating a new project, TeXnicCenter will analyse the directories specified in the path options. The sub folders of the template directories listed there will become categories, which are represented by the tabs on the left side of the dialog 'Project New' and the files contained in the sub folders will be listed, when choosing the according tab.

In our example, these tabs would be available:

  • Articles
  • General
  • Letters
  • University

The categories "Articles" and "General" would contain the templates from the company's directory. The category "University" would contain the templates from the local drive. The category "Letters" would contain the templates from the company's directory merged with the one's from the local drive:

  • "Order" (company/Letters/Order.tex)
  • "Notice" (company/Letters/Order.tex)
  • "Love Letter" (local/Letters/Love Letter.tex)

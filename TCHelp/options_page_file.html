Tab 'File'

The tab 'File' in the dialog 'Options' allows to modify the file handling of TeXnicCenter.

The meaning of the controls are:

'Save all before compilation'
If enabled, TeXnicCenter will save all modified files, before the build process is started.
'Save automatic every x minutes'
If enabled, TeXnicCenter will save all modified files at the specified interval.
'Default'
TeXnicCenter supports three file formats:
  • Windows
  • Unix
  • Macintosh
Those formats only differ in the character sequence used as line feed.

Choose the default format here. The format for each file can be changed in the dialog 'File|Save As...' individually.

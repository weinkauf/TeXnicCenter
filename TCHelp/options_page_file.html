Options - Page 'File'

The page 'File' of the 'Options' dialog allows you to modify TeXnicCenter's file handling.

The meaning of the controls are:

'Save all before compilation'
If this option is enabled, TeXnicCenter will save all modified files, before it starts the build process.
'Save automatic every x minutes'
If this option is enabled, TeXnicCenter will save all modified files in the specified interval.
'Default'
TeXnicCenter supports three file formats
  • Windows
  • Unix
  • Macintosh

Those formats only differ in the character sequence used as a line feed.

Choose the format which should be used by default here. You can change it for each file when saving it via 'File|Save As...'.

Project Management

TeXnicCenter provides a project management, that helps you to keep files of one document together and to generate output for a document - especially, when working on large documents.

For each project, a project file will be generated that keeps the project's properties. Those informations are:

  • The path of the project's main file (the file that will be passed to the LaTeX compiler), relative to the directory, the project file is placed in.
  • A flag that specifies, if the current project uses BibTeX.
  • A flag that specified, if the current project uses MakeIndex.

Some features like the structure view (navigator bar on the left side of TeXnicCenter's main window), and the automatic start of BibTeX and MakeIndex are only available, when a project is open.

Using projects is very simple. Just create a new project, when starting the work on a new document. If the requirements for the porject have changed, you can simply modify the project's properties.

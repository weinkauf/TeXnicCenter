Project Management

TeXnicCenter provides a project management tool, that helps to keep the files of one document together and to generate output for a document - especially, when working on large documents.

For each project, a project file will be generated, that stores the project's properties. Those informations are:

  • The path to the main file of the project (This file will be passed to the LaTeX compiler), relative to the directory, the project file is placed in.
  • A flag that specifies, if the current project uses BibTeX.
  • A flag that specified, if the current project uses MakeIndex.

Some features like Structure View (i.e. the navigator bar on the left side of the main window), and the automatic start of BibTeX and MakeIndex are only available, if a project is opened.

Using projects is very simple. Just create a new project, before starting to work on a new document. If the requirements for the project have changed, you can simply modify the project's properties.

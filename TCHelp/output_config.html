Configuration of the Output Profiles

To configure the output profiles, there are two ways:

  • The first and the simplest one (especially if you are using MikTeX), is to use the Ouput Wizard. This wizard will also be started automatically, if you are starting TeXnicCenter for the first time.
  • The second way is to configure the ouput profiles manually. This is more work intensive, but provides you also more flexibility.

Naturally you can configure the output profiles, generated by the output wizard, later manually, using the output configuration.

If you are using the MikTeX-distribution, you should always use the output wizard. He will recognize, that MikTeX is installed on your system and will generate output profiles for DVI, PDF and postscript.

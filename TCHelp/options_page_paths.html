Tab 'Directories'

The tab 'Directories' in the dialog 'Options' allows to modify directory paths used by TeXnicCenter.

The meaning of the controls are:

'Project Template Directories'
The directory were the template(s) of the project is/are stored.
'Document Template Directories'
The files in the directories listed here will be provided as document templates when a new document is created.
'Working Directory'
The directory specified here will be used as the default directory for new projects and new files. If this path is left empty, the default directory for the Windows user is used. If a project is still open, the project directory is used.

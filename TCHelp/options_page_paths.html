Options - Page 'Directories'

The page 'Directories' of the 'Options' dialog allows you to modify directory paths used by TeXnicCenter.

The meaning of the controls are:

'Project Template Directory'
'Document Template Directories'
The files in the directories listed here will be provided to the user as document templates when they create a new document.
'Default'
The directory specified here will be used as the default directory for new projects.

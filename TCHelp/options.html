Options

TeXnicCenter provides many options to configure the application, making it easier for you. You can change these options by opening the 'Options' dialog and choosing the menu item 'Tools|Options...'.

The dialog contains five pages:

  1. The page 'General'
  2. The page 'File'
  3. The page 'Directories'
  4. The page 'Spelling'
  5. The page 'Clean'

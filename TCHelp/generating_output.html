Generating Output

To generate an output file for printing (i.e. DVI, PostScript, etc.) output types need to specified, a new project has to be created or an existing project has to be opened.

Select the type of output TeXnicCenter should generated, by choosing from the combo-box in the toolbar 'LaTeX', or by selecting in 'Output|Select Output Type...'.

After all the configuration, choose 'Output|Build Output' to start the generation of the output file. TeXnicCenter starts the necessary compiler and prints the output of the compiler to the output bar at the bottom of the TeXnicCenter window.

After the compilation is done, TeXnicCenter displays the number of errors, warnings and bad boxes. Jump to those results by choosing one of the commands 'Next *'/'Previous *' in the menu 'Output'.

If no errors are displayed, choose 'Output|View Output' to start or refresh the viewer, specified in the output type configuration, for the current output type.

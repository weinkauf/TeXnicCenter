Customization

TeXnicCenter allows you, to customize nearly every aspect of the graphical user interface (GUI).

To customize the user interface of TeXnicCenter, open the customization dialog using the menu item 'Tools|Customize...'. Then you can

  • Specify, which commands should be available on the menus and toolbars.
  • Customize the toolbars.
  • Customize the 'Tools' menu.
  • Customize the keyboard shortcuts used to access commands using the keyboard.
  • Customize the menus.
  • Customize the text format of the editor, the navigator and the output bar.
  • Customize the general look & feel.

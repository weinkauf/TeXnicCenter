Creating a new Project

To create a new project, simply choose the menu item 'File|New Project...' and the following dialog will appear:

Choose one of the document templates from the list on the left side. The tabs are representing the available categories of the available templates. What templates are available, depends on you. Take a look on how to use document templates to learn how this works.

The other controls are having the following meanings:

'Project Name'
Name of the new project. This will also be the title of the project file and of the project's main file.
'Project path'
The directory, where the new project should be located. If the directory does not exist, it will be created automatically.
'File format'
Choose which line delimiter should be used for the newly generated files.

Take a look at the file options for detailed information.

'Uses BibTeX'
Enable this option, if TeXnicCenter should run BibTeX, when generating the output for this project.
'Uses MakeIndex'
Enable this option, if TeXnicCenter should run MakeIndex, when generating the output for this project.
'Template'
The text displayed here is a description for the currently selected document template.

Press the 'OK' button to generate the new project. TeXnicCenter will create the project file and the project's main file, based on the choosen template. Afterwards the newly created project will be opened in TeXnicCenter.

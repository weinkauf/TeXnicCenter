Options - Page 'General'

The page 'General' of the 'Options' dialog allows you to set TeXnicCenter's general options.

The meaning of the controls are:

'Restore last session on start up'
If this option is enabled, TeXnicCenter will reload the project and the files on start, which opened, when TeXnicCenter has been quit the last time.
'Replace quotation marks (")'
If this option is enabled, TeXnicCenter will replace opening quotation marks with the string specified by 'opening quotation mark' and closing quotation marks with the string specified by 'closing quotation mark', when the user types the quotation mark (") character.

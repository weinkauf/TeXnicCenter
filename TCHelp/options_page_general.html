Tab 'General'

The tab 'General' in the dialog 'Options' allows to set general options for TeXnicCenter'.

The meaning of the controls are:

'Restore last session on start up'
If enabled, TeXnicCenter will on start reload the project and the files, which were open before closing TeXnicCenter last time.
'Replace quotation marks (")'
If enabled, TeXnicCenter will replace opening quotation marks with the string specified by 'opening quotation mark' and closing quotation marks with the string specified by 'closing quotation mark', when the user types the quotation mark (") character.

Welcome to TeXnicCenter

  • Project orientated integrated development environment for (La)TeX documents
  • Define unlimited "output profiles" (i.e. "LaTeX => DVI", "LaTeX => PostScript", "LaTeX => PDF")
  • Editor:
    • Syntax highlighting
    • Dynamic word wrapping (No need to scroll horizontally!)
    • Integrated spell checker, includes real time spell checking (Checks during data input!)
    • Unlimited undo and redo
    • Bookmarks
    • History ("Go to last change")
    • Incremental search
    • Find / Find and replace
    • Fully customizable
  • Structure View:
    • Clear, document orientated tree view of the whole document structure
    • Displays headers, floating figures and floating tables
    • Updates by saving one or all files
    • Supports file links via \input and \include
    • Simple navigation by double clicking on an item
    • Simple referencing by inserting references with an item's context menu
  • LaTeX Support:
    • Simply inserti LaTeX constructs via menu or toolbar
    • Compile project in the IDE - jump to errors, warnings and bad boxes with a click of the mouse
    • Forward and inverse search supported, for viewers, which support these features
  • Supports document templates
  • Menu and toolbars fully customizable in modern and skinable look and feel
  • Language support for English and German

Options - Page 'Spelling'

The page 'Spelling' of the 'Options' dialog allows you to set TeXnicCenter's spelling and language options. Advanced configuration information is available for this option page.

The meaning of the controls are:

'Language'
This control contains the languages of the installed dictionaries. The language setting only affects the spell checking functionality of TeXnicCenter. It does not affect the language of TeXnicCenter. Changes to language do not take affect until the next time TeXnicCenter is started.
'Dialect'
This control contains the language dialects available for the selected language. The dialect setting only affects the spell checking functionality of TeXnicCenter. It does not affect the language of TeXnicCenter. Changes to dialect do not take affect until the next time TeXnicCenter is started.
'Locale'
This control selects the locale setting for the TeXnicCenter application. Locale information is used to format system messages and some internal text manipulation. Changing the locale setting does not change the language of TeXnicCenter. If the selected locale is not available on your Windows installation, the default system locale will be used.
'Check spelling while typing'
When selected, TeXnicCenter performs spell checking while you type. Words not found in the spell checker dictionary are highlighted.
'Suggest from main dictionary only'
When selected, TeXnicCenter produces suggestion lists based only on words from the main dictionary. Words in your personal dictionary are not included in suggestion lists.
'Ignore comments'
When selected, TeXnicCenter does not spell check comments.
'Ignore LaTeX tags'
When selected, TeXnicCenter does not spell check LaTeX tags. This setting does not affect spell checking LaTeX command arguments. For example, the argument of the LaTeX command \caption{This is a caption}, "This is a caption", is always spell checked. This option on only determines if "\caption" is spell checked.
'Ignore words with numbers'
When selected, TeXnicCenter does not spell check words containing numbers.
'Ignore UPPERCASE words'
When selected, TeXnicCenter does not spell check words containing all uppercase letters.
'Personal Dictionary'
This control contains the path and name of the file containing your personal dictionary. The personal dictionary contains words not found in the main dictionary. The personal dictionary is loaded once during application startup and written once during application shutdown. Clearing the personal dictionary text field disables the personal dictionary. Changes to personal dictionary control do not cause the personal dictionary to be reloaded.

Editing Output Profiles manually

To create new output profiles or to edit existing ones, you have to choose the menu item 'Build|Define Output Profiles...'. The following dialog will appear:

The 'Profiles' list shows you, which profiles are already defined. The four buttons at the bottom of the list are having the following meanings:

'Add'
Adds a new output profile to the list.
'Copy'
Copies the selected output profile to a new profile.
'Rename'
Allows you to change the name of the selected output profile.
'Remove'
Removes the selected output profile from the list.

The other buttons are performing the following actions:

'Wizard'
Starts the Profile Wizard that will help you generating new output profiles.
'OK'
Stores the changes and closes the dialog.
'Cancel'
Ignores the changes and closes the dialog.

To edit an output profile, select a profile in the list and modify the settings on the three pages.

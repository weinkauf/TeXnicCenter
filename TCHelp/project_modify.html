Modifying the Project Settings

The settings, that have been choosen, when the project was created, can be changed afterwards, by choosing the 'Project|Properties...' menu item. When choosing this item, the following dialog will appear:

The controls are having the following meanings:

'Main file'
This is the path of the project's main file, that should be passed to the (La)TeX compiler, when generating the output.

The path has to be specified relatively to the directory, the project file is placed in.

'Uses BibTeX'
Enable this option, if TeXnicCenter should run BibTeX, when generating the output for this project.
'Uses MakeIndex'
Enable this option, if TeXnicCenter should run MakeIndex, when generating the output for this project.

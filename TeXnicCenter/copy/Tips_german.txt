Willkommen beim TeXnicCenter!\pWir haben einige Tipps und Tricks zum Umgang mit LaTeX im Allgemeinen und TeXnicCenter im Speziellen für Sie parat.\pLassen Sie diese Dialogbox ruhig eine zeitlang aktiviert, bis Sie der Meinung sind, alle Features zu kennen und zu beherrschen. Wenn Sie diese Dialogbox nicht mehr sehen wollen, so deaktivieren Sie das Kästchen in der linken unteren Ecke. Sie können diesen Dialog jederzeit über den Menüpunkt '?|Tipps und Tricks...' aufrufen.
Aller Anfang ist schwer.\pWir haben ein paar Vorlagen für Sie erstellt, um den Anfang zu erleichtern. Öffnen Sie den Dialog zum Erstellen neuer Projekte ('Datei|Neues Projekt...') und probieren Sie die dort vorhandenen Vorlagen aus.
Um eine druckbare Datei (DVI, PS oder PDF) zu erzeugen, müssen Sie Ihr Dokument von LaTeX verarbeiten lassen. Das nennt man 'Kompilieren'.\pTeXnicCenter hilft Ihnen dabei. Im Menü 'Ausgabe' finden Sie alle Befehle zum Erzeugen und Betrachten der Ausgabedatei.
Am einfachsten ist die Arbeit mit TeXnicCenter, wenn Sie mit Projekten arbeiten.\pNeue Projekte erstellen Sie mit 'Datei|Neues Projekt...'\pFür vorhandene TEX-Dateien erstellen Sie ein Projekt, indem Sie die Datei zunächst laden und dann den Menüpunkt 'Projekt|Erzeugen mit aktueller Datei als Hauptdatei' wählen.
Sie können diverse LaTeX-Konstrukte und -Symbole einfach per Menü oder Tastenkürzel einfügen. Schauen Sie sich dazu mal die Menüs 'Einfügen' und 'Mathe' an...
Sie können Ihr LaTeX-Dokument in verschiedenen Formaten ausgeben lassen. Dazu gehören u.a. PDF, Postscript (PS) und DVI.\pUm das Ausgabeformat zu wechseln, wählen Sie den Menüpunkt 'Ausgabe|Aktives Ausgabeprofil wählen...'.\pDesweiteren finden Sie eine entsprechende Auswahlbox in der LaTeX-Symbolleiste.
Hinter den einfach anmutenden Ausgabeprofilen (z.B. "LaTeX => PDF") stehen recht komplexe Anweisungen für externe Programme wie den LaTeX-Compiler oder das jeweilige Anzeigeprogramm.\pSchauen Sie sich diese Anweisungen ruhig mal an: 'Ausgabe|Ausgabeprofile definieren...'.\pIn diesem Dialog finden Sie auch einen Assistenten, der die Standardkonfiguration wieder herstellt (wenn beim Ausprobieren mal was schief gegangen ist...).
Projekte, Projekte, Projekte!\pNur wenn Sie in TeXnicCenter mit Projekten arbeiten, kann eine Gliederung Ihres Dokumentes erstellt werden.\pErstellen Sie ein Projekt ('Datei|Neues Projekt...' oder 'Projekt|Erzeugen mit aktueller Datei als Hauptdatei') und aktivieren Sie die Gliederungsansicht mit 'Ansicht|Navigatorleiste'.
Die Gliederungsansicht am linken Rand des TeXnicCenter-Fensters stellt die aktuelle Struktur Ihres geöffneten Dokumentes dar.\pWenn Sie auf einen Eintrag doppelklicken, springt der Cursor zur entsprechenden Stelle in der entsprechenden Datei. Wenn Sie auf die rechte Maustaste klicken, wird ein Menü geöffnet, das Ihnen z.B. das Einfügen von Querverweisen erlaubt.\pDie Gliederungsansicht wird immer dann aktualisiert, wenn Sie eine geöffnete Datei speichern.
Entgegen hartnäckiger Gerüchte können Sie Sonderzeichen wie 'ä' (anstatt "a) oder 'ß' (anstatt "s) direkt in Ihrer TEX-Datei verwenden, wenn Sie in der Präambel Ihrer TEX-Datei die Anweisungen\p\usepackage[ngerman]{babel}\n\usepackage[ansinew]{inputenc}\pangeben.\pAnstelle von 'ngerman' können sie auch 'francais', 'polish', 'spanish', 'russian', 'italian', 'czech', 'klingon' usw. verwenden.
Die automatische Silbentrennung funktioniert in internationalen Texten mit Umlauten etc. nur dann korrekt, wenn LaTeX sogenannte T1-enkodierte Schriften zur Verfügung stehen.\pEs gibt verschiedene Wege zu diesem Ziel. Einer davon ist die Verwendung der Latin Modern Schriftart. Hierzu schreiben Sie folgende Befehle in die Präambel Ihres Dokuments:\p\usepackage[T1]{fontenc}\n\usepackage{lmodern}
TeXnicCenter ersetzt Anführungszeichen (") während des Schreibens automatisch durch deutsche öffnende bzw. schließende Anführungszeichen.\pWenn Sie andere Anführungszeichen wünschen oder diese Funktion deaktivieren möchten, so können Sie das unter 'Extras|Optionen...|Allgemein' tun.
Sie können Ihr LaTeX-Dokument in beliebig viele TEX-Dateien aufteilen um zu große und somit unübersichtliche Dateien zu vermeiden (z.B. für jedes Kapitel eine eigene Datei).\pFügen Sie dazu in der Hauptdatei für jede zu verwendene Unterdatei den Befehl '\input{Unterdatei}' ein. Das hat dann den gleichen Effekt, als wenn an der Stelle des \input-Befehls direkt der Inhalt der Datei stünde.
Sie können veranlassen, dass TeXnicCenter in regelmäßigen Abständen alle geöffneten Dateien unaufgefordert speichert.\pAktivieren Sie dazu den Punkt 'Automatisch speichern' unter 'Extras|Optionen...|Datei' und stellen Sie das gewünschte Speicherintervall ein.
Der Befehl 'Suchen|Gehe zur letzten Änderung' setzt den Cursor an die Stelle im aktuellen Editorfenster, an der Sie zuletzt Text eingegeben oder entfernt haben.\pDas ist praktisch, wenn Sie "nur mal eben" durch das Dokument geblättert haben um etwas nachzuschlagen und nun an der vorherigen Stelle mit dem Bearbeiten fortfahren wollen.
Wenn Sie Dokumente für Deutsche Papierformate (z.B. A4) entwickeln, sollten Sie speziell dafür entworfene Dokumentklassen verwenden.\pMögliche Klassen enthält z.B. das KomaScript-Paket (ftp://ftp.dante.de/).
Mit der Tastenkombination STRG+F2 können Sie Lesezeichen im Editor setzen.\pMit der Taste F2 können Sie zum nächsten Lesezeichen springen.\pEs gibt auch ein Fenster, in dem alle Lesezeichen angezeigt werden. Sie öffnen es mittels 'Ansicht|Hilfsfenster|Lesezeichen'. Dort können Sie Ihren Lesezeichen auch aussagekräftige Namen geben.
Um Grafiken in LaTeX-Dokumente einzubetten, müssen Sie die Anweisung\p"\usepackage{graphicx}"\pin der Präambel ihres Dokumentes einfügen. Mit dem Menüpunkt 'Einfügen|Grafik...' können Sie dann eine Grafik in das Dokument einbetten.\pDer nächste Tipp gibt noch weitere Anregungen zum Verwenden von Grafiken in LaTeX-Dokumenten...
Zum Zeichnen von Grafiken für LaTeX-Dokumente eignen sich alle Grafikprogramme, die Grafiken im PDF- oder EPS-Format exportieren können. Dazu gehören unter anderem die freien Programme Inkscape (www.inkscape.org) und OpenOffice.org Draw (www.openoffice.org).\pMarkieren Sie im jeweiligen Zeichenprogramm den relevanten Bereich und exportieren Sie diesen als PDF- oder EPS-Datei. Die so entstandene Datei kann über den Menübefehl 'Einfügen|Grafik...' in ihr LaTeX-Dokument eingebunden werden.
Probieren Sie zum Suchen von Textstellen mal die Funktion 'Suchen|Inkrementelle Suche' aus -- der zu suchende Text wird während des Tippens in der Statusleiste angezeigt und der gefundene Text im Editor ...
Wenn Textbereiche in der Ausgabe genauso aussehen sollen wie in Ihrer LaTeX-Datei (besonders geeignet für Programmcode), dann betten Sie ihn in eine Verbatim-Umgebung.\pAbgesetzten Text einfach mit "\begin{verbatim}" und "\end{verbatim}" umschließen.\pText innerhalb einer Zeile mit "\verb" einleiten. Das erste Zeichen danach gibt an, durch welches Zeichen das Argument beendet wird (z.B. "\verb|C:\test\test.tex|" um in der Ausgabe "C:\test\test.tex" zu erhalten).\pAlternativ können Sie den Befehl 'Einfügen|Code' verwenden.
Mit der Tastenkombination Strg+Tab können Sie zwischen den geöffneten Dokumenten wechseln.
Der Befehl 'Fenster|Fenster...' öffnet einen Dialog, der Ihnen alle geöffneten Fenster anzeigt und Ihnen das Schließen einzelner oder mehrerer Fenster ermöglicht.
Die Symbolleisten von TeXnicCenter können frei auf dem Bildschirm platziert werden. Klicken Sie dazu mit der Maus auf eine freie Stelle der Symbolleiste und ziehen Sie sie an eine beliebige Stelle.\pDas gleiche gilt sogar für die Menüleiste. Probieren Sie doch mal, die Menüleiste senkrecht am linken oder rechten Fensterrand anzudocken...
Sie können die Tastenkürzel für Befehle unter 'Extras|Anpassen...' frei vergeben.
Um Text kursiv zu setzen (hervorzuheben) sollten Sie ihn mit dem Kommando "\emph{}" umschließen.\pSie können den Text markieren und dann den Befehl 'Format|Zeichen|Hervorgehoben' auswählen, um diese Formatierung vorzunehmen.\pOder Sie drücken einfach STRG+E, wenn Sie den Text markiert haben.
Der Befehl 'Einfügen|Eigene Textbausteine|Textbausteine verwalten...' öffnet einen Dialog, in dem Sie eigene Textbausteine und Befehle erzeugen können. Klicken Sie dazu den Button 'Neu' an.
Sie können die Menü- und Symbolleisten von TeXnicCenter nach Ihren Anforderungen anpassen. Ziehen Sie dazu einfach einen Menü- oder Symbolleistenbutton mit gehaltener ALT-Taste auf eine beliebige andere Leiste oder wählen Sie 'Extras|Anpassen...'.
Die Navigatorleiste (Baumansicht am linken Rand) und das Ausgabefenster (am unteren Rand) können in der Größe geändert werden, wenn Sie mit der Maus den dicken Balken zwischen den Leisten und dem Editorfenster verschieben.\pSie können die Leisten auch vom Fensterrand abreißen, indem Sie mit der Maus an eine freie Stelle der Leiste klicken und sie dann auf eine beliebige Bildschirmposition ziehen.
Mit Hilfe des Befehls 'Suchen|Suchen in Dateien', können Sie in mehreren Dateien nach einem bestimmten Text suchen.\pDiese Funktion eignet sich besonders bei großen Dokumenten, die in viele Dateien zerlegt wurden, um eine bestimmte Textstelle zu finden oder kontinuierliche Fehler zu beheben.
Sie haben jetzt alle Tipps gesehen!

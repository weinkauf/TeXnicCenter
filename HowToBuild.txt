Building TeXnicCenter

1 Requirements

The following software and libraries are required to successfully build 
TeXnicCenter and all of its components:
  * Visual C++ 2008 with MFC libraries
  * Visual C++ 2008 Feature Pack, freely available at 
    http://www.microsoft.com/downloads
  * Microsoft HTML Help Workshop
  * MSXML 4

To generate the setup you will need:
  * Nullsoft Scriptable Install System (NSIS) available at 
    http://nsis.sourceforge.net/

2 Build Process
  
  1. Open the TeXnicCenter solution TeXnicCenter.sln in Visual C++.
  2. Build the Debug and/or Release versions of TeXnicCenter.



Visit the TeXnicCenter-Homepage

     http://www.texniccenter.org/

for further information. Don't hesitate to ask us, if a problem occurs.
